MD Chips!

  • Assist Chips - common actions, such as adding events to a calendar.
  • Filter Chips - used to filter contents, such as shopping catalogs.
  • Input Chips - information entered by the user, such as attendees to an event.
  • Suggestion Chips - represents dynamic suggestions for user inputs, such as message replies.

Chip Sets

Chips should always appear as a set. Chip sets are toolbars that can display any combination of chips or any other items.
